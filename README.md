# FairPay — Site institucional (Next.js 14, App Router, estático)

Este projeto é um site **institucional** (governança/credibilidade), **não comercial**.

## Rodar local
```bash
npm install
npm run dev
```

## Build estático
```bash
npm run build
```
O build gera a pasta `out/` (export estático).

## Imagens reais
Coloque as imagens em `public/images/`:
- `fachada-358.png`
- `escultura.png`
- `cafair.png`

Você pode substituir por `.jpg/.webp` mantendo os nomes.
